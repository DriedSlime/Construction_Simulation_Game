# 중장비 조작 시뮬레이션 게임

---

유니티(Unity) 엔진을 활용하여 제작한 건설 장비 시뮬레이션 기반의 미니게임 3종 패키지 프로젝트입니다. 
플레이어는 포크레인, 트럭, 크레인을 직접 조작하여 각 스테이지의 미션을 수행하고 최종 성적을 부여받습니다.

## 게임 플레이 및 핵심 루프
본 게임은 **철거 -> 운반 -> 조립**으로 이어지는 실제 건설 공정을 모티브로 유기적으로 연결되어 있습니다. 

1. **1단계: 철거 (포크레인 게임)** - 건물 및 구조물을 파괴하고 짐을 모두 옮겨 클리어합니다.
2. **2단계: 운반 (트럭 게임)** - 확보된 자재를 목적지까지 안전하게 운반합니다.
3. **3단계: 조립 (크레인 게임)** - 크레인을 조작하여 타겟 존(Zone) 안에 구조물을 정확하게 배치합니다.
4. **결과 화면 (EndScene)** - 각 스테이지의 점수가 연동되어 기록되며, 기록된 랭크(**S, A, B, C**)가 나열되는 것을 볼 수 있습니다.

---

## 개발 환경 (Environment)
- **Engine**: Unity 6000.3.5f2
- **Input**: Input System Package
- **UI, Font**: TextMeshPro, MapleStory Light, MapleStory Bold
- **Platform**: PC (Windows)
